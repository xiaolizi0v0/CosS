# CosS 2D 世界技术方案

##  引言

世界中每个角色NPC代表一个Agent，而一个Agent的内核是一个终端CLI。这个世界需要与之前的项目中智能体终端隔离，世界需要从0开始。世界中的角色的信息交流都是Agent自行发起的，世界系统的职责则是将Agent的信息给到另一个Agent，类似于中转站。在这个世界中，Agent都有自己的房子，用户可以在特定的地方左键创建角色，创建成功就会生成对应的角色的房子以及角色。世界场景中心有一个公告栏，用户可点击公告栏进行交互，发布任务，发布任务之后，世界中的角色会根据任务进行一论交流，明确自己在这次任务中的需要完成的模块，然后开始完成各自的模块，期间会出现角色们之间的多次交流，直到最后完成任务，在这之间角色的内核终端CLI需要开启完全访问权限，以免角色之间交流不流畅。在世界主页UI中有一个聊天图标，点击聊天图标后弹出聊天框可以查看各个角色们的聊天记录（UI制作成微信群聊样式）。

引擎：Phaser 3
