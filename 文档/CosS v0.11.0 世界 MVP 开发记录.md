# CosS v0.11.0 世界 MVP 开发记录

## 版本定位

v0.11.0 在 v0.10.0 Kernel 调度架构之上，引入新的「世界」概念，作为项目之外的第二类工作空间入口。

本版本目标是交付世界功能的 MVP：

1. 左侧栏在「项目」下方新增「世界」入口。
2. 点击「世界」后，左侧列表切换为世界列表。
3. 世界列表右上角 `+` 可新建世界。
4. 新建世界弹窗支持用户自定义世界名称，并选择保存文件夹。
5. 进入世界后显示一个 2D 卡通像素世界主页。
6. 世界主页中，每个像素小人代表一个角色 Agent。

## 状态模型

前端状态新增世界相关字段：

- `activeSidebarSection`：当前侧边栏主区域，支持 `projects` 和 `worlds`。
- `activeWorldId`：当前打开的世界 ID。
- `worlds`：世界列表。

单个世界的 MVP 数据结构包含：

- `id`：世界 ID。
- `name`：世界名称。
- `path`：世界保存文件夹。
- `createdAt` / `lastOpenedAt`：创建与最近打开时间。
- `terrain`：世界地形标识，当前默认 `pixel-meadow`。
- `agents`：世界中的角色 Agent 像素小人位置与状态。

## 左侧栏交互

- 项目列表与世界列表共用侧边栏列表区域。
- 点击「项目」时显示项目列表，并保持原有项目工作区行为。
- 点击「世界」时显示世界列表，并将主工作区切换为世界主页。
- 世界列表右上角加号打开「新建世界」弹窗。
- 创建世界后自动选择并打开新世界。

## 新建世界弹窗

新建世界弹窗包含：

- 世界名称输入框。
- 保存文件夹输入框。
- 文件夹选择按钮。
- 创建并打开按钮。

校验规则：

- 世界名称不能为空。
- 保存文件夹不能为空。
- 文件夹选择沿用现有 Electron 目录选择能力。

## 世界主页 MVP

世界主页是一个纯前端 2D 像素场景，包含：

- 天空、太阳、云朵。
- 草地网格背景。
- 小屋、树、水塘、道路等像素地图元素。
- 角色 Agent 像素小人。
- 世界说明卡片。

像素小人由现有角色模板生成，当前包含：

- 产品经理
- 技术负责人
- 前端工程师
- 后端工程师
- 测试工程师
- AI/Agent 工程师
- DevOps 工程师
- 技术文档工程师
- 安全工程师

## 后续演进方向

v0.11.0 仅完成世界 MVP。后续版本可继续扩展：

1. 世界持久化独立文件结构。
2. 世界内 Agent 状态与真实任务执行状态联动。
3. 世界地图编辑器。
4. Agent 在世界中的移动、对话和协作事件动画。
5. 世界与项目、任务、记忆之间的关联关系。
6. 不同世界模板和地形主题。

## 验证目标

本版本重点验证：

- 应用启动显示 `v0.11.0`。
- 点击左侧栏「世界」可切换到世界列表。
- 世界列表为空时显示空状态。
- 点击世界列表右上角 `+` 可打开新建世界弹窗。
- 新建世界时可填写名称和选择保存文件夹。
- 创建世界后主区域显示 2D 像素世界主页。
- 世界主页中可看到多个角色 Agent 像素小人。
- 原有项目列表、项目打开、任务窗口和 Kernel 相关流程不受影响。
