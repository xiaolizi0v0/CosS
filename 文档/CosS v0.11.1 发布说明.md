# CosS Desktop v0.11.1 发布说明

发布日期：2026-07-13

## 版本概览

v0.11.1 正式推出 Agent 世界。用户可以在像素小镇中查看 9 位 Agent 居民及其住宅，管理世界群聊成员，并通过公告栏发布协作任务。

## 主要更新

- 新增全屏 Agent 小镇场景，支持程序化地图、天空背景、林带、道路、装饰物与摄像机移动。
- 为 9 位内置居民提供独立住宅、房间场景、头像、休闲动画、奔跑动画与开门动画。
- 新增世界群聊成员管理。只有已加入群聊的居民会接收公告栏任务。
- 完成公告栏任务流程：居民从家中出门、前往公告栏领取任务、返回房间并开始工作。
- 新增居民详情与任务记录，可查看群聊状态、任务指令、协作消息和技术执行日志。
- 优化角色移动方向、公告栏站位、场景层级、房间内角色位置与提示样式。
- 统一 Agent 世界的中文与英文界面文案，移除面向用户的开发占位术语和重复系统前缀。
- 将地图渲染、地形生成与摄像机控制拆分为独立模块，降低单文件复杂度。

## 版本与兼容性

- 应用版本、工作流协议版本与项目状态版本统一升级至 `0.11.1`。
- GitHub Release 提供 Windows、Linux 与 macOS 构建产物，以及源码归档、SHA-256 校验和与签名文件。
- 执行真实的 Agent 世界任务前，请在「设置 > 智能体设置」中配置 CodeBuddy Code API Key。

## 升级建议

安装新版本前，建议通过「设置 > 数据管理」创建备份。升级后首次进入 Agent 世界时，系统会根据当前版本重新整理地图布局与居民住宅位置。

## 验证范围

本版本按照模块化测试策略完成验证，未运行全局测试套件：

- Agent 世界脚本语法检查。
- 世界地图、摄像机、地形、居民资源与房间场景单元测试。
- 世界群聊、成员范围、公告栏任务流、居民移动与房间位置定向端到端测试。
- v0.11.1 版本显示与协议版本一致性检查。
