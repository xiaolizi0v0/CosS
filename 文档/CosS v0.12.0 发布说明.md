# CosS v0.12.0 发布说明

## 重点更新

- 新增可执行蓝图工作区，支持节点库、流程/数据引脚、拖拽连线、对齐与分布、网格吸附、迷你地图、右键菜单、节点分组和折叠。
- 支持根据任务目标自动生成蓝图；用户可以在画布中继续微调，创建任务时填写本次运行目标。
- 蓝图运行面板支持节点状态、执行动画、事件日志、审批、人工输入、交付物和失败节点重跑。
- Agent 节点默认执行时限调整为 15 分钟。
- 补齐蓝图工作区与 Agent 世界地图的 UI 翻译；切换语言后会强制刷新 DOM 和地图引擎文本。
- 修复蓝图任务记录删除、文件夹选择、连线端点跟随鼠标和节点标题/引脚遮挡等问题。

## 发布校验

GitHub Actions 的 Release 工作流在 PR、标签发布和手动发布前，仅运行蓝图工作区专属语法检查与 Playwright 校验：

```powershell
npm run test:blueprint:syntax
npm run test:blueprint:release
```

发布预检覆盖蓝图翻译实时刷新、画布操作、任务图生成、多类型引脚和任务记录删除等关键路径；完整蓝图套件仍可通过 `npm run test:blueprint` 手动运行。

工作流通过后会使用 electron-builder 构建 Windows、Linux 和 macOS 安装包，并上传源码归档、签名文件与校验和。
