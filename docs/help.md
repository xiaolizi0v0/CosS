# CosS 帮助文档

## 简介

CosS（Code Orchestration System）是一款桌面端 AI 协作工作台，支持通过多 Agent 协作完成软件开发任务。它将 AI 能力与项目管理、任务编排深度整合，帮助团队高效交付。

## 快速开始

### 1. 安装

从 [GitHub Releases](https://github.com/xiaolizi0v0/CosS/releases) 下载最新版本的安装包，按照指引完成安装。

### 2. 创建项目

启动 CosS 后，在左侧面板点击「创建项目」，填写项目名称和描述即可创建新项目。

### 3. 定义角色

在项目设置中定义参与协作的角色。系统内置以下角色模板：

- **技术负责人** — 负责技术决策、架构设计和代码审查
- **产品经理** — 负责需求分析、功能规划和用户故事编写
- **后端工程师** — 负责后端服务和 API 开发
- **前端工程师** — 负责前端界面开发
- **测试工程师** — 负责测试用例编写和质量保证

### 4. 创建任务

点击「创建任务」按钮，输入任务描述。系统会自动将任务拆解为可执行的工作步骤并分配给对应角色。

### 5. 开始协作

任务启动后，各 Agent 角色会按工作流顺序执行。你可以在桌面视图中实时观察每个角色的执行进度和输出。

## 项目管理

### 创建项目

在项目列表页点击「新建项目」，设置项目名称和描述。每个项目包含独立的任务板、Agent 角色和交付物记录。

### 项目设置

在设置页中，你可以：

- 管理项目角色
- 配置 Agent 模型偏好
- 查看项目任务板
- 管理交付物记录

### 删除项目

在项目设置中选择「删除项目」。注意：删除操作不可撤销，项目所有数据将被清除。

## Agent 协作

### 工作流

CosS 使用线性工作流（Linear Workflow）来协调多 Agent 协作。每个任务按照预设步骤顺序执行，前一步完成后自动触发下一步。

### 任务分配

Agent 从中央任务队列获取任务。系统根据角色定义自动分配任务给合适的 Agent。

### 实时监控

你可以在桌面视图中实时查看：

- 每个 Agent 的当前状态（空闲/运行中/已完成）
- Agent 的执行日志和输出
- 任务进度条

## 设置与配置

### 通用设置

- **语言** — 支持中文、英文、日文、韩文
- **主题** — 亮色/暗色模式切换
- **字体大小** — 调整界面文字大小

### AI 模型配置

在「模型设置」面板中，你可以配置：

- 默认使用模型（如 DeepSeek）
- API 密钥
- 模型参数（温度、最大 Token 等）

### 会话管理

- 查看历史会话
- 导出会话记录
- 清除会话数据

### 日志管理

在帮助菜单中点击「打开日志目录」即可查看应用日志，用于排查问题。

## 常见问题 (FAQ)

### Q: Agent 没有响应怎么办？

检查网络连接是否正常，以及 API 密钥配置是否正确。如果问题持续，请查看应用日志并提交反馈。

### Q: 如何更新 CosS？

从 GitHub Releases 页面下载最新版本覆盖安装即可。配置数据会自动保留。

### Q: 任务卡住了如何重新执行？

在任务板中找到对应任务，点击「重新执行」按钮即可重新运行该任务。

### Q: 如何保护我的数据隐私？

CosS 的所有数据处理均在本地完成。详细说明请参阅[隐私保护声明](privacy.md)。
